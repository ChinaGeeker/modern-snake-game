# 🐍 现代霓虹风格贪吃蛇 (Modern Snake Game)

一款基于 HTML5 Canvas 开发的视觉惊艳、体验流畅的网页版贪吃蛇游戏。

![项目预览](https://raw.githubusercontent.com/ChinaGeeker/modern-snake-game/main/screenshot.png) *(注：如果您有截图可以替换此路径)*

## ✨ 核心特性

- **视觉效果**: 采用现代暗黑模式、毛玻璃 (Glassmorphism) 风格、动态霓虹渐变以及平滑的微动画。
- **用户系统**: 支持本地存储的用户注册、登录和个人会话管理。
- **排行榜系统**: 实时追踪个人最高分，并展示全局前 10 名排行榜。
- **响应式设计**: 完美适配桌面端与移动端浏览器。
- **性能优化**: 使用高效的 Canvas 渲染和模块化 JavaScript 开发。

## 🕹️ 游戏控制

- **PC端**: 使用方向键 (`↑` `↓` `←` `→`) 或 `WASD` 键控制蛇的方向。
- **暂停**: 游戏中点击“暂停”按钮或按键盘上的特定键。

## 🛠️ 技术栈

- **前端**: HTML5, Vanilla CSS3 (Custom Variables / HSL), JavaScript (ES6+ Modules)
- **存储**: LocalStorage (持久化本地数据)
- **图形**: Canvas API

## 🚀 快速开始

1. 克隆本项目到本地：
   ```bash
   git clone https://github.com/ChinaGeeker/modern-snake-game.git
   ```
2. 直接在浏览器中打开 `index.html` 即可开始游戏。
